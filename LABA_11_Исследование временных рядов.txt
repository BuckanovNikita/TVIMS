Временные ряды

1. Изучить основные свойства временных рядов.

2. Сгенерировать временной ряд с трендом и сезонной составляющей. Добавить случайную погрешность для каждого значения ряда. Разбить выборку на две неравные части: одну для анализа, другую для проверки предсказаний. 

3. По анализу части выборки определить параметры временного ряда и выполнить предсказание. Проверить его на другой части выборки. Анализ выболнить для "зашумленного" и исходного рядов.
